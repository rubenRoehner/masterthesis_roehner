\subsection{Demand Forecasting Module}\label{sec:implementation_demand-forecasting-module}
The Demand Forecasting Module to feed accurate demand forecasts into the RL agents depends on the implementation by Li et al. \cite{li_improving_2023}.
This demand forecasting is build upon an Irregular Convolutional Long Short Term Memory (IrConvLSTM) network.
The core of this model are the Irregular Convolution layers, which replace the standard kernels of a Convolutional layer with fixed spatial kernels that link semantic neighbors.
